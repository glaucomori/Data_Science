# Alfabeto
___

Desafio

Dada a letra N do alfabeto, nos diga qual a sua posição.

___
Entrada

Um único caracter N, uma letra maiúscula ('A'-'Z') do alfabeto (que contém 26 letras).

___
Saída

Um único inteiro, que representa a posição da letra no alfabeto.
